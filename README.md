# go-backend-template

## Dependencies

golang >= 1.24.4
air >= v1.62.0 (install with `go install github.com/air-verse/air@latest`)

## Commands

```sh
make dev-api # requires air to be installed

make dev-cli # requires air to be installed

make build-api

make build-cli

make run-api

make run-cli

make clean
```
